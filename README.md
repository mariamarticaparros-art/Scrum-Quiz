# 🚀 Scrum Mastery Learning Tool

Esta aplicación es una herramienta interactiva diseñada para el aprendizaje y la consolidación de conocimientos sobre la **Guía Scrum 2020**. Permite a los usuarios estudiar la teoría fundamental y poner a prueba sus conocimientos mediante un sistema de cuestionarios altamente personalizable.

## 📋 Funcionalidades Principales

### 1. Centro de Conocimiento (Teoría)
La aplicación incluye una sección dedicada a la **Guía Scrum 2020**, permitiendo al usuario revisar los fundamentos, pilares y eventos del marco de trabajo.

### 2. Quiz Personalizable
Desde el menú principal, el usuario tiene control total sobre su sesión de aprendizaje:
- **Modo de Juego:** Puedes elegir entre realizar un cuestionario genérico o enfocarte específicamente en tus **preguntas de repaso** (aquellas marcadas previamente por el usuario).
- **Parámetros de Sesión:** Ajusta fácilmente la cantidad de preguntas y el tiempo límite (minutos) mediante deslizadores intuitivos.
- **Orden de Aparición:** Elige entre un modo **Secuencial** (para seguir un orden lógico) o un modo **Aleatorio** para poner a prueba tu conocimiento bajo presión.

### 3. Gestión de Aprendizaje
- **Preguntas de Repaso:** El sistema permite identificar y marcar preguntas con una estrella (⭐) para facilitar el estudio enfocado.
- **Punto de Inicio:** En el modo secuencial, puedes seleccionar una pregunta específica del listado como punto de partida para tu sesión de estudio.
